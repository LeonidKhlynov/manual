**2. Работа с приложением**
===========================

В этой части руководства, основное внимание будет уделено работе с самим приложением. Приложение хранит информацию по театральной системе, данные из которой можно **удалять** или **добавлять**.

* **Иконка дома** - ссылка на Главную страницу

* **Иконка гаечного ключа** - ссылка для администрирования Web-приложения

.. important:: Доступ к данной ссылке имеет только администратор! Если вы не являетесь администратором, дальнейшая работа с данной функцией будет невозможна.

* **Шары** - обозначают таблицы базы данных различных частей театральной системы

.. figure:: static/4.png
	:align: center
	
	Рис.1 - Главная страница

При нажатии на любой из "Шаров", приложение перенесет вас на старинцу с подробным описанием таблицы и её значений. На примере перейдем в таблицу "Гастроли".

.. figure:: static/5.png
	:align: center
	
	Рис.2 - Таблица гастролей

Информация выводится на экран в виде карточек, каждая карточка состоит из:
	* Названия
	* Номера
	* Кнопки удаления карточки
	.. important:: Удаление карточек возможно лишь при наличии прав администратора!

	* Атрибута (Левая часть карточки)
	* Значения (Правая часть карточки)

.. figure:: static/6.png
	:align: center
	
	Рис.3 - Обзор карточки

Помимо просмотра карточек, в возможности пользователя входит и добавления своих карточек в таблицу. Для этого требуется нажать на иконку плюса в правом верхнем углу.

.. figure:: static/7.png
	:align: center
	
	Рис.4 - Добавление карточки
